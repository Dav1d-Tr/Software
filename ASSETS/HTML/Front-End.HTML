Front-End

What is a Front-End developer?

A Front end Developer is the person in charge of programming a web browser, that is to say, he is in charge of translating the design and visual style definitions made in previous stages to semantic HTML codes. It is one of the fundamental pieces in a digital project.

Skills

Skills

All frontend developers should have knowledge of HTML, CSS and JavaScript. Those three languages are the foundation for almost everything you'll do. You'll need other skills as well, but the same skill set required will vary from job to job.
